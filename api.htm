请求地址：{$smarty.const.AURL}/api

参数1：返回数据类型

参数名：type

可取值：(json|text|js)

默认值：json

参数2：回调函数(JSONP)

参数名：callback

可取值：任意字符

默认值：无

参数3：要查询的ip地址

参数名：ip

可取值：域名或ipv4

默认值：访者ip

示例1：查询用户ip及其地理位置，返回text
{$smarty.const.AURL}/api?type=text
示例2：查询指定为61.177.36.4的地理位置，返回json，并设置回调函数为chkip
{$smarty.const.AURL}/api?ip=61.177.36.4&callback=chkip
示例3：查询指定为www.baidu.com的服务器地理位置，并返回json
{$smarty.const.AURL}/api?ip=www.baidu.com
-本站API支持易于调用的jsonp格式和txt格式作为信息源。
-查询效率高，每秒可执行4k次查询，平均每次查询10ms（不计算网络延时）。
